Код на языке VBA для CorelDraw версии 22 и выше. 
В начале программы объявлены единицы измерения миллиметры, чтобы избежать перерасчетов при работе.
Скрипт собирает в массив координаты X и Y центров находящихся на текущем слое объектов. 
Координаты указываются в миллиметрах, округляются до целых значений миллиметра по обычным правилам округления. 
Далее массив сортируется для прохождения по объектам змейкой. Обход начинаем от от левого нижнего угла вправо по X. 
После сортировки массив нормируется относительно координат левого нижнего объекта, его мы принимаем за начало координат, координаты остальных объектов рассчитываются относительно него. 
Итоговый массив сохраняетя в текстовый файл с тем же названием и в ту же папку, что и файл CorelDraw.
